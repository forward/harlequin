Harlequin.js

Harlequin.js is a javascript utility for applying colour highlighting to data tables, useful for showing the distribution of data. Simply include the library in your page and you're off.

View the README or the example usage to find out how to start colouring!

Custom PainterUsing a custom painter to decorate the cells

                    
  Harlequin.register("mypainter",function(cell,min,max,sort,color,options){

    var size = Math.floor(((cell.value-min)/(max-min)) * 14) + 12;
    var grey = 200 - Math.floor(((cell.value-min)/(max-min)) * 200);
    cell.el.css("font-size",size+"px")
           .css("color","rgb("+grey+","+grey+","+grey+")")
           .css("border-bottom","1px solid "+color);

  });

  // .stripe("...") is actually an alias for .paint("stripe",...), so 
  // you call custom painters methods on a table by passing the painter
  // name prior to the rest of the options

  Harlequin.paint("mypainter","custompainter","row");
